LearnSphere is a large-scale cloud-based learning platform designed to simulate how modern distributed applications operate in real environments. The system is built using a microservices architecture where each service is independently deployed and managed, allowing the platform to support complex features such as course management, user authentication, video streaming, analytics, and real-time communication.

A major focus of this project was automation and operational design. The platform includes a serverless media pipeline that automatically processes uploaded videos into multiple streaming formats. The infrastructure is fully provisioned using Infrastructure as Code, enabling reproducible deployments and consistent environments.

This project demonstrates hands-on experience with container orchestration, CI/CD automation, cloud infrastructure design, observability practices, and distributed system management.
