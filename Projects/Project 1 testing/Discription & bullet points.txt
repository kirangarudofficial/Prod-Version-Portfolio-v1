LearnSphere is a cloud-based learning platform developed as a real-world deployment simulation to understand how modern distributed systems operate in controlled environments. The platform is built using a microservices architecture where each service is independently containerized and deployed, enabling modular development of features such as user management, course delivery, video processing, analytics, and system monitoring.

The infrastructure is designed for a single AWS region and single Availability Zone to replicate real deployment workflows while keeping resource usage optimized for testing purposes. This setup allowed experimentation with cluster orchestration, networking, deployment automation, and operational monitoring without requiring full high-availability configurations.

A key focus of this project was automation and platform behavior validation. The system includes an event-driven media processing pipeline that automatically handles video uploads, triggers serverless processing workflows, and manages metadata updates. All infrastructure components are provisioned using Infrastructure as Code, enabling repeatable environment setup and teardown.

This project demonstrates practical hands-on experience with container orchestration, CI/CD automation, cloud infrastructure provisioning, observability integration, and distributed system deployment within a controlled testing environment.
